# Super Memo 2 for LC
Implementing super memo 2 for leet code style problems

## Modification to the Algo
Instead of checking which day the user must practice the problem, the algo returns the order to practice the problems. Helps user not feel pressured mentally to do the days problems and just pick it back up whenever they want :).
